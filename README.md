## 3 功能模块

项目的目标是实现一个图书商城的基础功能，可以供不同用户使用。需要以下功能的实现：

- 用户的注册、登录

- 进行图书的增加、删除、修改、查询（后台与前台）

- 客户按照页码浏览查询的图书列表

- 客户按照价格区间查询图书并且分页

- 购物车功能

- 订单功能
